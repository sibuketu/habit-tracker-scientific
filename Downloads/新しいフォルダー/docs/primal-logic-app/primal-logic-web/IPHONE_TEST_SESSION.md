## ✅ テスト項目（優先度順）

### 1. 初回起動・初期設定

- [ ] **同意画面（Consent）**: 同意画面が表示されるか
- [ ] **オンボーディング**: オンボーディング画面が表示されるか
- [ ] **初期画面**: ホーム画面が表示されるか

### 2. ホーム画面（Home）

- [ ] **画面表示**: ホーム画面が正しく表示されるか
- [ ] **栄養素ゲージ**: 栄養素ゲージが表示されるか（ビタミンC、K、鉄、ナトリウムなど）
- [ ] **ゲージタップ**: 栄養素ゲージをタップしてArgument Cardが表示されるか
- [ ] **P:F比率ゲージ**: P:F比率ゲージが表示されるか
- [ ] **下部ナビゲーション**: 下部ナビゲーションが表示されるか（ホーム、入力、履歴、その他、設定）

### 3. 入力画面（Input）

- [ ] **画面遷移**: 「入力」タブをタップして入力画面に遷移できるか
- [ ] **食品検索**: 食品検索ボックスで検索できるか（例: 「牛肉」）
- [ ] **食品選択**: 検索結果から食品を選択できるか
- [ ] **量の入力**: 食品の量（g）を入力できるか
- [ ] **食品追加**: 「追加」ボタンで食品を追加できるか
- [ ] **栄養素計算**: 追加後に栄養素が計算されるか
- [ ] **手動入力**: 手動入力機能が動作するか

### 4. 栄養素表示

- [ ] **ゲージ更新**: 食品追加後、ホーム画面のゲージが更新されるか
- [ ] **ゾーン表示**: 4ゾーングラデーションが正しく表示されるか
  - Zone 1: 不足（赤）
  - Zone 2: やや不足（オレンジ）
  - Zone 3: 十分（緑）
  - Zone 4: 過剰（青）
- [ ] **Argument Card**: 栄養素をタップしてArgument Cardが表示されるか
- [ ] **Argument Card展開**: Argument Cardを展開して詳細が表示されるか

### 5. AIチャット

- [ ] **AIチャット開く**: ホーム画面のAIチャットボタン（💬）をタップできるか
- [ ] **チャットUI表示**: チャットUIが表示されるか
- [ ] **メッセージ送信**: メッセージを送信できるか（例: 「カーニボアダイエットとは？」）
- [ ] **AI応答**: AIが応答を返すか
- [ ] **Todo表示**: AI応答にTodoが含まれる場合、Todoカードが表示されるか
- [ ] **チャット閉じる**: チャットを閉じられるか

### 6. 履歴画面（History）

- [ ] **画面遷移**: 「履歴」タブをタップして履歴画面に遷移できるか
- [ ] **履歴表示**: 過去の食事履歴が表示されるか
- [ ] **期間選択**: 期間選択（今日、週、月、全期間）が動作するか
- [ ] **統計表示**: 統計情報が表示されるか
- [ ] **グラフ表示**: グラフが表示されるか（あれば）

### 7. その他画面（Labs）

- [ ] **画面遷移**: 「その他」タブをタップしてLabs画面に遷移できるか
- [ ] **Bio-Tuner**: Bio-Tunerボタンが表示されるか
- [ ] **日記機能**: 日記機能が表示されるか
- [ ] **習慣トラッカー**: 習慣トラッカーが表示されるか
- [ ] **Doctor Defense**: 医者への説明ツールが表示されるか
- [ ] **Logic Armor**: Logic Armor（理論武装）が表示されるか
- [ ] **Tips機能**: Tips機能が表示されるか

### 8. 設定画面（Settings）

- [ ] **画面遷移**: 「設定」タブをタップして設定画面に遷移できるか
- [ ] **プロフィール設定**: プロフィール設定が表示されるか
- [ ] **栄養素目標値設定**: 栄養素目標値の設定が表示されるか
- [ ] **表示設定**: 表示設定（栄養素の表示/非表示）が動作するか
- [ ] **デバッグモード**: デバッグモードのON/OFFが動作するか（あれば）

### 9. プロフィール設定（Profile）

- [ ] **画面遷移**: プロフィール設定画面に遷移できるか
- [ ] **基本情報入力**: 性別、年齢、活動レベルなどを入力できるか
- [ ] **保存**: 設定を保存できるか
- [ ] **目標値更新**: 設定保存後、ホーム画面の目標値が更新されるか

### 10. データ保存・読み込み

- [ ] **データ保存**: 食品追加後、ブラウザをリロードしてもデータが残っているか
- [ ] **データ読み込み**: アプリを閉じて再度開いた時、データが読み込まれるか
- [ ] **複数日のデータ**: 複数日分のデータが正しく保存・表示されるか

### 11. UI/UX

- [ ] **レスポンシブ表示**: iPhoneの画面サイズに適切に表示されるか
- [ ] **タッチ操作**: ボタンやリンクがタッチしやすいサイズか（44px以上）
- [ ] **スクロール**: 画面がスクロールできるか
- [ ] **戻るボタン**: 戻るボタンが正しく動作するか（あれば）
- [ ] **ローディング表示**: データ読み込み中のローディング表示が適切か

### 12. エラーハンドリング

- [ ] **ネットワークエラー**: ネットワークエラー時に適切なメッセージが表示されるか
- [ ] **入力検証**: 不正な入力（例: 負の数、空文字）が拒否されるか
- [ ] **空データ表示**: データがない場合の表示が適切か

### 13. パフォーマンス

- [ ] **初回読み込み**: 初回読み込みが3秒以内か
- [ ] **画面遷移**: 画面遷移が1秒以内か
- [ ] **食品検索**: 食品検索が2秒以内に完了するか
- [ ] **AI応答**: AI応答が10秒以内に返ってくるか

### 14. Safari特有の確認

- [ ] **アドレスバー**: アドレスバーが正しく表示されるか
