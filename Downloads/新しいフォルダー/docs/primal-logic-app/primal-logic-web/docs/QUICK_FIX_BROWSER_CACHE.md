# ブラウザキャッシュ問題の即座解決方法

**作成日**: 2026-01-25  
**問題**: UIが変わらない（ブラウザキャッシュが原因）  
**解決時間**: 約30秒

---

## 即座に解決する手順（方法3不要）

### Step 1: 開発者ツールでキャッシュを無効化（最重要）

1. **ブラウザで `F12` を押す**（開発者ツールを開く）
2. **Network タブをクリック**
3. **"Disable cache" にチェックを入れる** ⭐ これが最重要
4. **開発者ツールは開いたままにしておく**

### Step 2: ハードリロード

- **`Ctrl+Shift+R`** を押す（Windows）または **`Cmd+Shift+R`** (Mac)

これで完了です。栄養ゲージが表示されるはずです。

- **方法3は不要**: 他のアプリのログイン情報を消す必要はありません
