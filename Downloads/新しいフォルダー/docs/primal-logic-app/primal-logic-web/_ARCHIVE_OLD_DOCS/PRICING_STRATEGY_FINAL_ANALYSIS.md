## 年収が減る理由

### $19.99の場合
- **獲得数**: +10ユーザー/月（$14.99の+15ユーザー/月より少ない）
- **平均単価**: ¥2,250/月（$14.99の¥1,950/月より高い）
- **年間収益**: 約122万円（$14.99の約162万円より少ない）

### 理由
- **高価格帯のため獲得数が限定的**: 価格が高いと、新規ユーザーの獲得数が減少
- **クーポン使用率**: 高価格帯ではクーポン使用率が高くなるが、それでも獲得数が限定的
- **収益の計算**: ユーザー数 × 平均単価 = 収益。ユーザー数の減少が平均単価の上昇を上回る
