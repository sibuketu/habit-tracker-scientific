# .envファイルの開き方（Cursorで表示する方法）

## 方法1: エクスプローラーから

1. `primal-logic-app/primal-logic-web` フォルダをエクスプローラーで開く
2. `.env` ファイルを右クリック → 「Cursorで開く」（または「プログラムから開く」→ Cursor）

## 方法2: Cursor内から開く

1. Cursorで `primal-logic-web` フォルダを開いている状態で
2. **Ctrl+P**（Windows）または **Cmd+P**（Mac）でクイックオープン
3. `.env` と入力してEnter

## 方法3: ファイル一覧から探す

1. Cursorの左サイドバーのファイルツリーで `primal-logic-web` フォルダを展開
2. `.env` ファイルを探す（通常はフォルダの最上部に表示される）
3. クリックして開く

## APIキーの貼り付け位置

`.env` ファイルを開いたら、**4行目**の `your_gemini_api_key_here` の部分を実際のAPIキーに置き換えてください。

```env
VITE_GEMINI_API_KEY=your_gemini_api_key_here  ← この部分を実際のAPIキーに置き換え
```

**例:**
```env
VITE_GEMINI_API_KEY=AIzaSyC...（実際のAPIキー）
```

**重要:**
- `your_gemini_api_key_here` の部分を削除して、実際のAPIキーを貼り付け
- `VITE_GEMINI_API_KEY=` の `=` の後ろに貼り付ける
- スペースや改行は入れない

## 注意事項

- `.env` ファイルが存在しない場合は、新規作成してください
- `.env` ファイルは `.gitignore` に含まれているため、Gitにはコミットされません
- APIキーなどの機密情報を含むため、共有しないでください

## ファイルが表示されない場合

`.env` ファイルが表示されない場合は、以下を確認してください：

1. Cursorの設定で「隠しファイルを表示」が有効になっているか
2. ファイルが実際に存在するか（エクスプローラーで確認）
3. `.env.example` があれば、それをコピーして `.env` を作成
