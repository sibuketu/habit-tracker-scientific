# MCP (Model Context Protocol) 使用メモ

## 将来の用途（現在は不要）

### UI参考資料の取得
- MCPを使ってUIデザインの参考になるサイトをスクレイピング
- 類似アプリのUIパターンを分析
- デザインシステムの収集

### 外部AI機能の統合（DeepResearchなど）
- **DeepResearch**: カーニボア専門家の最新の動画、記事、研究を検索・分析
- **Web検索**: 最新の研究論文や専門家の主張を検索
- **YouTube検索**: ケンベリー、ショーン・ベイカーなどの最新動画から情報を取得

### 使用タイミング
- UI改善が必要になった時
- デザインパターンの研究が必要な時
- 参考になるアプリを分析したい時
- **カーニボア専門家の最新情報を取得したい時**
- **特定の研究論文やデータを検索したい時**

### 注意事項
- 現在のプロンプト拡張は十分包括的（専門家の思考パターン、推論方法、口調まで反映）
- ただし、最新情報や特定の研究を参照する必要がある場合は、MCPでDeepResearchやWeb検索を使用
- 使用する際は、ユーザーに確認してから実装する

### 現在の拡張レベル評価
- ✅ **十分**: 専門家の思考パターン、推論方法、口調まで反映済み
- ✅ **十分**: 知識ベース（knowledgeBase.ts、remedyLogic.ts）に主要な情報が含まれている
- ⚠️ **要検討**: 最新の研究や動画の情報が必要な場合は、MCPでDeepResearchを使用
