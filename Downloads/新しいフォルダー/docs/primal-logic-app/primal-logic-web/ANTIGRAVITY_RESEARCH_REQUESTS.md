# AntiGravity リサーチ依頼

## 1. カーニボア移行期間の詳細リサーチ

### ファイル名
`CARNIVORE_TRANSITION_PERIOD_RESEARCH.md`

### リサーチ内容
カーニボアダイエットの移行期間に関する包括的なリサーチ：

1. **移行期間の長さ**
   - 専門家（Dr. Ken Berry、Dr. Shawn Baker、Dr. Paul Saladino、Dr. Anthony Chaffee、Dr. Bart Kay、Dr. Paul Mason）の見解
   - 個人差の要因（体質、健康状態、移行前の食事、年齢、性別など）
   - 平均的な移行期間と範囲（例：20-40日、30-60日など）

2. **移行期間中の症状**
   - 症状の種類と頻度
   - 症状の原因（電解質不足、代謝適応、消化器系の適応など）
   - 症状の対処法（栄養素調整、推奨食品、説明文）

3. **移行期間の進捗指標**
   - どのような指標で移行期間の進捗を判断するか
   - 症状の改善、エネルギーレベルの変化、睡眠の質など

4. **個人差を考慮した計算方法**
   - 30日固定ではなく、個人差を考慮した柔軟な計算方法
   - 移行期間の開始日・終了日の設定方法

5. **移行期間中の栄養素目標値の自動調整**
   - 症状に応じた栄養素目標値の調整方法
   - 例：頭痛→ナトリウム+2000mg、こむら返り→マグネシウム+200mg

### 情報源
- Dr. Ken Berry（YouTube、書籍、ブログ）
- Dr. Shawn Baker（YouTube、書籍、ブログ）
- Dr. Paul Saladino（YouTube、書籍、ブログ）
- Dr. Anthony Chaffee（YouTube、書籍、ブログ）
- Dr. Bart Kay（YouTube、書籍、ブログ）
- Dr. Paul Mason（YouTube、書籍、ブログ）
- カーニボアコミュニティの実体験（Reddit、Facebookグループなど）

### 出力形式
Markdown形式で、以下のセクションを含む：
- 移行期間の長さ（専門家の見解、個人差の要因、平均的な期間と範囲）
- 移行期間中の症状（種類、原因、対処法）
- 移行期間の進捗指標
- 個人差を考慮した計算方法
- 移行期間中の栄養素目標値の自動調整

2. **UI/UX設計**

   - ユーザーに機能を選択させる方法

3. **技術的な実装**
