# Antigravity用: App Store アプリアイコン作成プロンプト

## 📱 要件

**App Store Connect用アプリアイコン**
- **サイズ**: 1024 × 1024px（必須）
- **形式**: PNG
- **アルファチャンネル**: なし（透明部分なし）
- **角丸**: なし（Appleが自動で角丸を適用）
- **ファイル名**: `app-icon-1024.png`

---

## 🎨 デザイン要件

### アプリ情報
- **アプリ名**: CarnivOS
- **コンセプト**: 世界初のカーニボア（肉食）ダイエット専用アプリ
- **ブランドイメージ**: 科学的、原始的、力強い、健康的

### 既存アイコンデザインの参考
プロジェクト内に以下のアイコンが存在：
- `icon-steak-fire.png`: ステーキと炎のデザイン
- `icon-spear-gold.png`: 金色の槍のデザイン
- `icon-v2-512.png`: バージョン2のデザイン

これらのデザイン言語を参考にしつつ、App Store用に最適化されたアイコンを作成してください。

### デザインの方向性

#### 必須要素
1. **CarnivOSのブランドアイデンティティ**を反映
2. **肉食ダイエット**を連想させる要素（肉、炎、原始的な力）
3. **科学的・論理的**な印象（「Logic」の部分）
4. **モダンで洗練された**見た目（App Storeで目立つ）

#### 推奨要素
- 肉のシルエット（リブロース、ステーキなど）
- 炎や原始的な力の表現
- 「C」や「OS」の文字を組み込む（控えめに）
- グラデーションやシャドウで立体感
- 明るい背景でも暗い背景でも見やすいコントラスト

#### 避けるべき要素
- 複雑すぎるデザイン（小さいサイズで見えなくなる）
- 文字が多すぎる（「CarnivOS」のフルテキストは避ける）
- グロテスクな表現（App Store審査で問題になる可能性）
- 既存アプリと似すぎたデザイン

---

## 🎯 作成依頼

以下のバリエーションを作成してください：

### バリエーション1: ステーキ + 炎
- リブロースステーキのシルエット
- 背景に炎のエフェクト
- 「C」の文字を控えめに組み込む

### バリエーション2: 槍 + グラデーション
- 金色の槍のシルエット
- 背景は深い赤からオレンジへのグラデーション
- 原始的な力の表現

### バリエーション3: ミニマル + 文字
- シンプルな「C」の文字
- 肉のテクスチャやパターンを背景に
- モダンで洗練された印象

### バリエーション4: 抽象的な肉のシルエット
- 肉の断面やマーブル模様を抽象化
- 科学的な印象（分子構造のようなパターン）
- 「Logic」の部分を強調

### バリエーション5: 組み合わせデザイン
- 上記の要素を組み合わせたオリジナルデザイン
- 最もApp Storeで目立つデザイン

---

## 📋 出力形式

各バリエーションについて：
1. **1024 × 1024pxのPNGファイル**（アルファチャンネルなし）
2. **ファイル名**: `app-icon-1024-v1.png`, `app-icon-1024-v2.png` など
3. **デザインの説明**（各バリエーションの特徴）
