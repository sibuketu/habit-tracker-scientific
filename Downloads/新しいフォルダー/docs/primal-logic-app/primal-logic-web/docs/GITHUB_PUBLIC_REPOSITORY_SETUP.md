# GitHubリポジトリをPublicにする方法

## 手順

1. GitHubにログイン: https://github.com
2. リポジトリ「CarnivoreGuid」を開く: https://github.com/sibuketu/CarnivoreGuid
3. リポジトリの「Settings」タブをクリック
4. 左サイドバーの「General」を選択
5. ページ下部の「Danger Zone」セクションまでスクロール
6. 「Change repository visibility」をクリック
7. 「Change visibility」をクリック
8. 「Make public」を選択
9. リポジトリ名「sibuketu/CarnivoreGuid」を入力して確認
10. 「I understand, change repository visibility」をクリック

## 確認

- リポジトリのトップページで「Public」バッジが表示されることを確認
- Netlifyでリポジトリが表示されることを確認
