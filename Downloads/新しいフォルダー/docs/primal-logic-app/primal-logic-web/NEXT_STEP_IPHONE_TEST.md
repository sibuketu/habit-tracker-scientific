## 📱 次のステップ: iPhoneからアクセスする準備

### 1. Windowsファイアウォール設定（必須）

iPhoneからアクセスするには、Windowsファイアウォールでポート5174を開く必要があります。

#### 方法1: GUIで設定（推奨）

1. Windowsの検索で「ファイアウォール」を検索
2. 「Windows Defender ファイアウォール」を開く
3. 「詳細設定」をクリック
4. 「受信の規則」→「新しい規則」
5. 「ポート」を選択 → 「次へ」
6. 「TCP」を選択 → 「特定のローカル ポート」に `5174` を入力 → 「次へ」
7. 「接続を許可する」を選択 → 「次へ」
8. すべてのプロファイルにチェック（ドメイン、プライベート、パブリック）→ 「次へ」
9. 名前を「Vite Dev Server (5174)」などに設定 → 「完了」

#### 方法2: PowerShellで設定（管理者権限）

PowerShellを**管理者権限で開き**、以下を実行：

```powershell
New-NetFirewallRule -DisplayName "Vite Dev Server (5174)" -Direction Inbound -Protocol TCP -LocalPort 5174 -Action Allow
```

詳細は `FIREWALL_SETUP.md` を参照してください。

## 🔧 トラブルシューティング

### iPhoneからアクセスできない場合

1. **ファイアウォール設定を確認**: 上記の手順でポート5174が開いているか確認

3. **開発サーバーの起動確認**: PCのブラウザで `http://localhost:5174` にアクセスできるか確認
