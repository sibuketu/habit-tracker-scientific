# 栄養素に関する質問項目 - 優先度順（100項目）

## 優先度の基準
- **変動量の大きさ**: 栄養素目標値への影響度が高い順
- **実用性**: 実際に測定・把握しやすい順
- **カバー範囲**: より多くの栄養素に影響する順

## 【最優先】必須項目（栄養素目標値に直接影響、変動量が大きい）

1. **性別** - 鉄分（8mg→18mg）、タンパク質の必要量が大きく変わる
2. **活動量** - タンパク質（+20%）、脂質（+20%）、マグネシウム（+100mg）に影響
3. **年齢** - ビタミンD（+1000IU）、タンパク質（+10g）に影響（50歳以上）
4. **ストレスレベル** - マグネシウム（+100mg）に影響
5. **睡眠時間** - マグネシウム（+50mg）、コルチゾールに影響（7時間未満）
