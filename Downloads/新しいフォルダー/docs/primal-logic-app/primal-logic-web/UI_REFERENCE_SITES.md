# UIリファレンスサイト

## チャットUIデザインの参考サイト

### 1. Mobbin
- **URL**: https://mobbin.com
- **説明**: 400種類以上のスマホアプリ、100,000パターンのUIがまとめられたギャラリー
- **用途**: 最新のモバイルアプリUIデザインパターンを確認

### 2. Dribbble
- **URL**: https://dribbble.com
- **説明**: 世界中のデザイナーが作品を共有するプラットフォーム
- **用途**: チャットアプリのデザインインスピレーション

### 3. CallToInspiration
- **URL**: https://coliss.com/articles/build-websites/operation/work/small-details-for-ui-calltoinspiration.html
- **説明**: WebサイトやスマホアプリのUIデザインのアイデアを集めたギャラリー
- **用途**: UIの細かいディテールやベストプラクティスを確認

### 4. UI Patterns
- **URL**: https://ui-patterns.com
- **説明**: UIパターンのギャラリー
- **用途**: チャットUIのパターンを確認

### 5. Pttrns
- **URL**: https://pttrns.com
- **説明**: モバイルアプリのUIパターン集
- **用途**: モバイルチャットUIのパターンを確認

## 参考にすべきポイント

1. **シンプルで機能的なUI**: 余計な装飾を避け、機能を優先
2. **モバイルファースト**: スマホでの操作性を最優先
3. **既存のUIを壊さない**: 新しいUIを追加する際は、既存の機能を維持
4. **段階的な改善**: 一度に全てを変更せず、段階的に改善

## 注意事項

- **GeminiのUIをそのまま採用しない**: アプリの目的に合わせてカスタマイズ
- **不要な機能を追加しない**: Deep Researchなど、アプリに不要な機能は実装しない
- **既存の機能を確認**: 新しいUIを追加する前に、既存の機能が動作しているか確認
