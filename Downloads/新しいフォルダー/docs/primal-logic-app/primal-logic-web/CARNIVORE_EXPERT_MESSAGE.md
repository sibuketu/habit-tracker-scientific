# Primal Logic - Carnivore専門家向けメッセージ

Carnivoreダイエットに精通されている皆様へ

この度、**Primal Logic**というカーニボアダイエット管理アプリのデモ版を開発いたしました。専門家の皆様にご覧いただき、ご意見やご指摘をいただければ幸いです。

## アプリの概要

**Primal Logic**は、カーニボアダイエットを実践する方々のための健康管理アプリです。Dr. Ken Berry、Dr. Shawn Baker、Dr. Paul Saladino、Dr. Anthony Chaffee、Dr. Bart Kay、Dr. Paul Masonなどの専門家の知見を参考に、栄養素の追跡と健康状態の管理をサポートします。

## 主な機能

### 1. 栄養素追跡
- **P:F Ratio（タンパク質：脂質比率）**のリアルタイム計算
- 主要栄養素の摂取量追跡（タンパク質、脂質、鉄分、マグネシウム、ビタミンD、ナトリウムなど）
- カーニボア基準に基づく栄養目標値の設定
- 4-zoneグラデーションゲージによる視覚的な栄養状態の表示

### 2. 習慣トラッカー
- 連続記録日数の追跡
- カレンダー形式での記録可視化
- フェーズ（称号）システム（Rookie → Carnivore → Hunter → Predator → Master → Legend）

### 3. 食品データベース
- 動物性食品の詳細な栄養情報
- 部位別の栄養素データ（例：リブアイ、ブリスケット、サーロインなど）
- カスタム食品の登録機能
- AIによる栄養素推測機能（食品名から自動推測）

### 4. 違反検知とリカバリー
- 植物性食品や加工食品の摂取を検知
- リカバリープロトコルの自動生成
- AIを活用したパーソナライズされたリカバリー提案

### 5. Logic Armor（理論武装機能）
- カーニボアダイエットの科学的根拠を表示
- 栄養素ごとの説明カード
- 反論に備えた知識の整理

## 技術的な特徴

### カーニボア基準に基づく栄養計算
- **タンパク質**: 体重1kgあたり1.2-2.2g（活動レベルに応じて調整）
- **脂質**: タンパク質の1.5-2倍（高脂質推奨）
- **炭水化物**: 実質ゼロ（植物性食品は検知対象）
- **微量栄養素**: カーニボア医者の推奨値に基づく

### 科学的根拠の重視
- カーニボア専門家の主張に基づく栄養目標値
- エビデンスベースの機能設計
- 推測ではなく、確立された知見を優先

## デモ版の制限事項

現在のデモ版には以下の制限があります：

1. **デバッグモード**: テスト用のデバッグデータを使用（設定画面で有効化）
2. **UI改善の余地**: デザインの統一感やプレミアム感の向上が必要
3. **機能の一部未実装**: 動画投稿、多言語対応などは今後の実装予定
