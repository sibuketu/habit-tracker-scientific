# iOS開発のコスト比較（Windows環境から）

## PWAとApp Storeの関係

### PWA（Progressive Web App）
- ✅ **App Store不要**: App Store審査なしで配布可能
- ✅ **無料**: App Storeの年会費（$99/年）不要
- ✅ **すぐに利用可能**: 審査待ちなし
- ✅ **iOS Safari対応**: 「ホーム画面に追加」でアプリのように利用可能
- ❌ **App Storeで公開不可**: App Storeには出せない
- ❌ **ネイティブ機能の制限**: 一部のネイティブ機能にアクセスできない場合がある

### App Store公開が必要な場合
- **App Storeで公開したい**: macOS環境が必要
- **ネイティブ機能をフル活用したい**: macOS環境が必要
- **App Storeの審査を通したい**: macOS環境が必要

## クラウドmacOSサービスの価格

### 1. MacinCloud

**料金プラン**:
- **Managed Server Plans**: 月額$25〜（約3,750円〜）
  - プリインストールされた開発ツール（Xcode等）を利用可能
  - 24時間アクセス可能
  - 専用サーバー
- **Pay-As-You-Go Plan**: 1時間$1〜（約150円/時間）
  - 使用時間に応じて課金
  - 使用頻度が低い場合に適している
  - 30時間のクレジットを前払い

**メリット**:
- 月額固定で予算が立てやすい
- セットアップが簡単
- リモートデスクトップでMacにアクセス

**デメリット**:
- サポートの対応が遅い場合がある（4営業日かかった例も）
- サーバーのパフォーマンスに不満の声も
- ネットワーク接続が必要

**評判**:
- ✅ 「セットアップが非常に簡単」「GitHubからのダウンロードやTestFlightへのアップロードが非常に速かった」
- ❌ 「サポートがなく、サーバーが応答しなくなった」「技術サポートがサーバーを再起動するのに4営業日かかった」

### 2. MacStadium

**料金プラン**:
- エンタープライズ向け（高価格）
- 詳細は要問い合わせ

**メリット**:
- エンタープライズ向けの高品質サービス
- 安定性が高い

**デメリット**:
- 個人開発者には高価

### 3. AWS EC2 Mac

**料金プラン**:
- 時間課金（使用した分だけ支払い）
- 詳細はAWSの料金ページを確認

**メリット**:
- 必要な時だけ利用できる
- AWSのインフラを活用

**デメリット**:
- 設定が複雑
- 時間課金のため、長時間使用すると高額になる可能性

## リモートビルドサービスの価格

### 1. EAS Build（Expo Application Services）

**料金プラン**:
- **Freeプラン**: 月5回まで無料
  - iOS/Androidビルドが月5回まで
  - OTA（Over-The-Air）アップデートのユーザー数制限（1000ユーザー）
- **有料プラン**: 月額約3,000円
  - ビルド回数の制限が緩和
  - より多くのビルドが可能

**メリット**:
- Expoプロジェクトの場合、簡単に利用できる
- 無料プランで試せる
- クラウドで自動ビルド

**デメリット**:
- Expoのエコシステムに依存
- 無料プランではビルド回数に制限がある
- 既存のプロジェクトとの互換性を確認する必要がある

### 2. Visual Studio App Center

**料金プラン**:
- 無料プランあり
- 有料プランは要確認

**メリット**:
- Microsoftのサービス
- CI/CD機能も含む

**デメリット**:
- 設定が複雑な場合がある

| サービス | 月額料金 | 特徴 |

| **EAS Build（無料）** | 無料 | 月5回までビルド可能 |

## 推奨アプローチ

### 1. 今すぐ（PWAで配布）
- **コスト**: 無料
- **方法**: PWA対応済みなので、そのまま配布可能
- **制限**: App Storeで公開できない

### 2. 短期間の開発（EAS Build無料プラン）
- **コスト**: 無料（月5回まで）
- **方法**: Expoプロジェクトの場合、EAS Buildでクラウドビルド
- **制限**: ビルド回数に制限がある

### 3. 本格的な開発（MacinCloud）
- **コスト**: 月額$25〜（約3,750円〜）
- **方法**: クラウドmacOSでXcodeを使用
- **メリット**: 24時間アクセス可能、専用サーバー

### 4. 長期的な開発（Mac購入）
- **コスト**: 初期費用10万円〜（Mac mini）
- **方法**: ローカルMacで開発
- **メリット**: 最も確実、高速、安定

## 結論

- **PWAで配布**: 無料、App Store不要、すぐに利用可能
- **App Store公開が必要**: macOS環境が必要（MacinCloud月額$25〜、またはMac購入10万円〜）
- **短期間の開発**: EAS Build無料プラン（月5回まで）
- **本格的な開発**: MacinCloud（月額$25〜）またはMac購入（初期費用10万円〜）
