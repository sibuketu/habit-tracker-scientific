### パーソナライズ版（推奨）

投資家ごとに以下をカスタマイズ：
- その投資家の投資テーマ・関心領域
- 過去の投資実績・ポートフォリオ

- その投資家が重視するポイント

【投資依頼】

## 修正点と原因

### 1. 「動画生成機能」の記述を削除
**原因**: 
- 投資家へのメッセージでは、アプリの機能に集中すべき
- 動画生成機能はSNS戦略（コンテンツマーケティング）の話であり、アプリ本体の機能ではない
- 投資家は「アプリが何ができるか」を知りたいのであって、「SNSで動画を作る機能」は関係ない

**修正**: 
- 投資家へのメッセージから「動画生成機能」の記述を削除
- アプリの機能（栄養素追跡、AIチャット、写真解析）に集中

### 2. 「日本独自の肉文化（業スー/和牛）」を削除
**原因**: 
- ゴールは「世界一のカーニボア健康アプリ」であり、日本限定の話ではない
- 投資家はグローバルな市場を考えるので、日本限定の話は意味がない
- アプリもグローバル展開を目指すべき

**修正**: 
- 投資家へのメッセージから「日本独自の肉文化」を削除
- アプリ内でもこのアイデアを抹消（コード内で使用されていないことを確認済み）

### 3. カーニボアダイエットとは何かを明確に説明
**原因**: 
- 投資家がカーニボアダイエットを知らない可能性がある
- アプリの価値提案を理解してもらうために、まずカーニボアダイエットとは何かを説明する必要がある

**修正**: 
- カーニボアダイエットとは何かを明確に説明
- 科学的根拠と専門家を明記
- 簡潔で投資家が理解しやすい内容に

### 4. アプリURLを追加
**原因**: 
- 投資家が実際にアプリを確認できるようにするため
- デモを見てもらうことで、投資家の理解が深まる

**修正**: 
- アプリURLを追加（NetlifyのURLなど）
- 「デモ: [アプリURL]」として明記
