# ブラウザキャッシュ問題の解決方法（方法3不要）

**作成日**: 2026-01-25  
**問題**: UIが変わらない（ブラウザキャッシュが原因）

---

## 解決方法（方法3は不要）

### 方法1: 開発者ツールでキャッシュを無効化（推奨）

1. **ブラウザで `F12` を押して開発者ツールを開く**
2. **Network タブを開く**
3. **"Disable cache" にチェックを入れる**（重要）
4. **開発者ツールを開いたまま、ページをリロード**（`Ctrl+R`）

この方法なら、他のアプリのログイン情報は消えません。

---

### 方法2: ハードリロード

- **`Ctrl+Shift+R`** (Windows) または **`Cmd+Shift+R`** (Mac)

この方法も、他のアプリのログイン情報は消えません。

---

### 方法3: 完全なキャッシュクリア（不要）

- この方法は他のアプリのログイン情報も消えるため、**使用しないでください**

---

## 開発サーバーの再起動

### 1. すべてのNodeプロセスを停止

```powershell
Get-Process | Where-Object { $_.ProcessName -like "*node*" } | Stop-Process -Force
```

### 2. 開発サーバーを再起動

```powershell
cd "C:\Users\susam\Downloads\新しいフォルダー\docs\primal-logic-app\primal-logic-web"; npm run dev
```

---

## 確認事項

再起動後、以下を確認してください：

1. **開発者ツールで "Disable cache" にチェックを入れる**
2. **ページをリロード**（`Ctrl+R`）
3. **栄養ゲージが表示されることを確認**
4. **"Stop Fasting" が正しく表示されることを確認**

---

## 注意事項

- **開発者ツールは開いたままにしておく**: "Disable cache" のチェックが外れると、再びキャッシュが有効になります
- **開発サーバーを再起動**: 古いプロセスが残っていると、変更が反映されません
