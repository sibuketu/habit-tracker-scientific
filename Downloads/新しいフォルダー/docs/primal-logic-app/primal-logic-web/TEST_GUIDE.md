# テストガイド

## テストの進め方

### 誰がテストする？
- **実機テスト（iPhone）**: ユーザーが手動でテスト（AIは物理的にiPhoneを操作できないため）
- **コードレビュー・自動テスト**: AIが実行可能

### テスト項目リスト

- `TEST_ITEMS_29_ONWARD.md`: 29以降のテスト項目（92項目）

### 進捗報告方法

番号で報告してください：
- 「29はOK、30は問題あり（チャットが閉じられない）」
- 「31-45は全部OK」
- 「46-60はまだ」

問題があれば、番号と問題内容を報告してください。AIが修正します。
