# 通知設定ガイド

## 通知が拒否されている場合の設定方法

### Windows（Edge/Chrome）

1. **ブラウザの設定を開く**
   - Edge: 設定（右上の三点メニュー）→ Cookieとサイトの権限 → 通知
   - Chrome: 設定（右上の三点メニュー）→ プライバシーとセキュリティ → サイトの設定 → 通知

2. **通知を許可するサイトを追加**
   - 「許可」セクションに `http://localhost:5174` を追加
   - または「通知を許可できるサイトを確認する前に確認する」を有効にする

3. **アプリを再読み込み**
   - ブラウザをリロード（F5）して、通知の許可を再度リクエスト

### Mac（Safari/Chrome）

1. **Safariの場合**
   - Safari → 環境設定 → ウェブサイト → 通知
   - `http://localhost:5174` を「許可」に設定

2. **Chromeの場合**
   - Chrome → 環境設定 → プライバシーとセキュリティ → サイトの設定 → 通知
   - `http://localhost:5174` を「許可」に設定

### 注意事項

- 本番環境では、HTTPSが必要な場合があります（多くのブラウザで通知APIはHTTPS必須）
- 開発環境（localhost）ではHTTPでも動作しますが、ブラウザによっては制限があります
