## UI改善（栄養素表示の多様化）

### 未実装のUIタイプ（Geminiとの会話から）

現在は**ゲージ（バー型）**のみ実装済み。以下のUIタイプを追加検討：

1. **レーダーチャート（6角形）**
   - PFCバランスや栄養の偏りを可視化
   - RPGのステータス画面風
   - バランス重視の栄養素に適している

2. **リング（ドーナツチャート）**
   - Apple Watchのアクティビティリング風
   - 省スペースでスタイリッシュ
   - 一覧性・お洒落さ重視

3. **グリッド（カード型）**
   - ビタミン・ミネラルをカード形式で一覧
   - コンプリート感を演出
   - 多品目管理（ビタミン類）に適している

4. **アバター（ボディマップ）**
   - 栄養が体のどこに効くかを可視化
   - 筋肉、臓器などが光る演出
   - 育成感重視

### 実装方針

- UIとロジックは既に分離されているため、コンポーネントの交換だけで実装可能
- 設定画面でUIタイプを選択可能にする
- 段階的に実装（Phase 2以降）

### 参考

- Geminiとの会話で提示された画像を参考に実装
- リリース後にアプリのスクリーンショットをGeminiに送り、代替案を提示してもらう
