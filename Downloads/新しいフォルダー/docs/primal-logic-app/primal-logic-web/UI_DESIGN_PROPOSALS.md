# UI設計案 - 機能配置と表示ON/OFF戦略

## 提案A: ホームシンプル + その他に全部（推奨）

### コンセプト
- **ホーム画面**: 最小限の機能のみ（P:F Ratio、主要栄養素ゲージ、いつものタブ）
- **その他（Labs）**: 全ての追加機能を配置、各機能にON/OFFスイッチ

### ホーム画面の機能（固定、ON/OFF不可）
- P:F Ratioゲージ
- 主要栄養素ゲージ（タンパク質、脂質、鉄分、マグネシウム、ビタミンD、ナトリウム）
- 「いつもの」タブ（よく使う食品）
- AI Speed Dial（写真・チャット・音声）

### その他（Labs）画面の機能（全てON/OFF可能）
- Logic Armor（理論武装）
- 習慣トラッカー
- Tips（豆知識）
- アプリアイコン生成
- **詳細プロファイル設定**（100項目の質問）
- **栄養素詳細分析**（全栄養素の詳細表示）
- **血液検査値入力**（オプション）
- **症状チェッカー**（詳細版）
- **食事パターン分析**
- **栄養素トレンドグラフ**

### メリット
- ホーム画面がシンプルで使いやすい
- 必要な機能だけをONにして使える
- OFFにしてもUIが崩れない（その他画面内で完結）

### デメリット
- その他画面が混雑する可能性

---

## 提案B: タブ分け + 各タブにON/OFF

### コンセプト
- **ホーム**: 最小限（P:F Ratio、主要栄養素ゲージ）
- **入力**: 食品入力、Status入力
- **履歴**: ログ履歴
- **プロファイル**: 基本設定 + 詳細設定（ON/OFF可能）
- **その他**: Labs機能（ON/OFF可能）

### 各タブのON/OFF機能

#### ホーム画面
- 「いつもの」タブ表示（ON/OFF）
- 栄養素プレビュー（ON/OFF）
- 詳細栄養素表示（ON/OFF）

#### プロファイル画面
- 基本設定セクション（常時表示）
- 推奨項目セクション（ON/OFF）
- 任意項目セクション（ON/OFF）
- 詳細項目セクション（ON/OFF）
- 血液検査値セクション（ON/OFF）

#### その他（Labs）画面
- Logic Armor（ON/OFF）
- 習慣トラッカー（ON/OFF）
- Tips（ON/OFF）
- アプリアイコン生成（ON/OFF）
- その他の実験的機能（ON/OFF）

### メリット
- 機能が整理されている
- 各画面で必要な機能だけをONにできる
- OFFにしてもUIが崩れない（セクション単位で非表示）

### デメリット
- 設定が複雑になる可能性

## 提案C: 段階的表示 + 折りたたみ

### コンセプト
- **ホーム**: 最小限（P:F Ratio、主要栄養素ゲージ）
- **プロファイル**: 段階的に展開できるセクション構成
- **その他**: Labs機能を折りたたみ可能に

### プロファイル画面の構成

```
基本情報（常時表示）
├─ 性別、年齢、活動量、ストレスレベル

推奨項目（折りたたみ可能、デフォルト展開）
├─ 睡眠時間、運動強度、運動頻度、甲状腺機能、日光暴露頻度

任意項目（折りたたみ可能、デフォルト折りたたみ）
├─ 消化器系、炎症レベル、メンタルヘルス、サプリメント、アルコール、カフェイン

詳細項目（折りたたみ可能、デフォルト折りたたみ）
├─ 体組成、ホルモン、症状、疾患、血液検査値

女性特有（折りたたみ可能、女性の場合のみ表示）
├─ 妊娠、授乳、閉経、PMS、PCOS等
```

### メリット
- 必要な項目だけを展開できる
- デフォルトで折りたたまれているのでUIがすっきり
- OFF機能と折りたたみの両方で柔軟に対応

### デメリット
- 折りたたみの状態管理が複雑

---

## 提案D: カテゴリ別タブ + 各カテゴリにON/OFF

### コンセプト
- **ホーム**: 最小限（P:F Ratio、主要栄養素ゲージ）
- **プロファイル**: カテゴリ別タブで分ける

### プロファイル画面のタブ構成

```
基本設定タブ（常時表示）
├─ 性別、年齢、活動量、ストレスレベル

ライフスタイルタブ（ON/OFF可能）
├─ 睡眠、運動、日光暴露、勤務スケジュール

健康状態タブ（ON/OFF可能）
├─ 消化器系、炎症、メンタルヘルス、症状

疾患タブ（ON/OFF可能）
├─ 自己免疫疾患、消化器疾患、代謝疾患

血液検査タブ（ON/OFF可能）
├─ 各種血液検査値

女性特有タブ（ON/OFF可能、女性の場合のみ）
├─ 妊娠、授乳、閉経、PMS、PCOS等
```

### メリット
- カテゴリが明確で分かりやすい
- 必要なタブだけをONにできる
- OFFにしてもUIが崩れない（タブが非表示になるだけ）

### デメリット
- タブが多くなりすぎる可能性

---

## 推奨: 提案A + 提案Cのハイブリッド

### コンセプト
- **ホーム**: 最小限（P:F Ratio、主要栄養素ゲージ、いつものタブ）
- **プロファイル**: 提案Cの折りたたみ方式 + 各セクションにON/OFF
- **その他**: 提案Aの方式（全てのLabs機能をON/OFF可能）

### 実装方針
1. **ホーム画面**: シンプルに保つ（追加機能は一切入れない）
2. **プロファイル画面**: 
   - 基本情報は常時表示
   - 推奨項目は折りたたみ可能（デフォルト展開）
   - 任意項目は折りたたみ可能（デフォルト折りたたみ）
   - 詳細項目は折りたたみ可能（デフォルト折りたたみ）
   - 各セクションに「このセクションを非表示にする」ON/OFFスイッチ
3. **その他（Labs）画面**: 
   - 全ての機能にON/OFFスイッチ
   - OFFにした機能は完全に非表示（UIが崩れない）

### メリット
- ホーム画面がシンプル
- プロファイル画面が整理されている
- OFFにしてもUIが崩れない
- ユーザーが自分好みにカスタマイズできる

### デメリット
- 実装がやや複雑
