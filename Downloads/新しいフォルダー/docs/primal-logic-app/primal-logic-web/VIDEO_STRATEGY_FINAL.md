# 動画生成戦略（最終版）

## ゴール

**世界一のCarnivoreアプリ**だけでなく、**世界一のCarnivoreチャンネル**を目指す。

---

## 基本戦略: 最初は解説動画に集中

### 方針

1. **最初は解説動画で行く**
   - 情報を小出しにせず、一気に解説動画で情報を提供
   - エンタメ動画は後から織り交ぜる
   - ただし、最初は解説動画に集中

2. **その後の展開**
   - 解説動画で基盤を固めた後、エンタメ動画も織り交ぜる
   - ただし、情報を小出しにするのではなく、各動画で完結した情報を提供

### 理由

- **情報の小出しは避ける**: ユーザーが「もっと知りたい」と思っても、次の動画まで待たなければならないのは不満
- **一気に解説**: 各動画で完結した情報を提供することで、ユーザーの満足度が高い
- **信頼性の構築**: 解説動画で専門性を示し、その後エンタメで親しみやすさを追加

## テンプレート選択ロジック

### ロング動画（YouTube用）

**デフォルト: 解説動画スタイル**
- **説明動画・教育動画**: 「Explainer Video」（最優先）
- **反論動画**: 「Breaking News」
- **体験談**: 「Explainer Video」（体験談も解説動画スタイルで統一）

### ショート動画（TikTok/Instagram用）

- **エンタメ・体験談**: 「Social Template」
- **反論動画**: 「Breaking News」
- **解説動画**: 「Social Template」（解説もSNS向けに最適化）

## 自動判定について（簡単に説明）

### コンテンツタイプの自動判定とは？

**スクリプトの中身を見て、どんな動画か自動で判断する機能**

例：
- スクリプトに「反論」「批判」という言葉がある → 「反論動画」と判定 → 「Breaking News」テンプレートを選ぶ
- スクリプトに「体験」「結果」という言葉がある → 「体験談」と判定 → 「Explainer Video」テンプレートを選ぶ
- 特にキーワードがない → 「説明動画」と判定 → 「Explainer Video」テンプレートを選ぶ（デフォルト）

### 動画タイプの自動判定とは？

**プラットフォームを見て、ロングかショートか自動で判断する機能**

例：
- `platform: 'youtube'` を指定 → 自動的に「ロング動画」と判定 → YouTube用のテンプレートを選ぶ
- `platform: 'tiktok'` を指定 → 自動的に「ショート動画」と判定 → TikTok用のテンプレートを選ぶ

**つまり**: ユーザーが「YouTube用の動画を作りたい」と言えば、自動的にロング動画用のテンプレートが選ばれる

## 複数選択肢の提示ルール

**Cursorが何か勧める時は、上からおすすめ順で複数アイデアを出す**

例：
1. **選択肢A（推奨）**: 英語メイン + 多言語対応
   - 理由: グローバルに広がる、リーチが広い
2. **選択肢B**: 日本語メイン + 英語版も作成
   - 理由: 日本語ユーザーに直接届く
3. **選択肢C**: 英語のみ（シンプル）
   - 理由: 実装が簡単

---

## 言語選択

**推奨: 英語メイン + 多言語対応**

- 英語版をメインに作成
- 同じスクリプトを日本語、フランス語、ドイツ語に翻訳して作成
- 各言語版を適切なプラットフォームに投稿

---

## 動画生成のワークフロー

### 1. ロング動画を作成（解説動画）

1. **スクリプト生成**（5-10分用、英語、解説動画スタイル）
2. **テンプレート選択**: 「Explainer Video」（自動選択）
3. **動画生成**: HeyGenでロング動画を生成
4. **保存**: YouTube用として保存

### 2. ロング動画からショート動画を作成（切り抜き）

1. **ロング動画を分析**: 最もインパクトのある部分を特定
2. **ショート動画用スクリプト生成**: 30-60秒用に要約（解説スタイルを維持）
3. **テンプレート選択**: 「Social Template」（自動選択）
4. **動画生成**: 同じ内容を短くしたバージョンを生成
5. **保存**: TikTok/Instagram用として保存

---

## 次のステップ

1. **解説動画を10本作成**（基盤構築）
2. **エンタメ動画を織り交ぜる**（親しみやすさを追加）
3. **反論動画を作成**（批判への対応）
4. **体験談動画を作成**（実践者の声）
