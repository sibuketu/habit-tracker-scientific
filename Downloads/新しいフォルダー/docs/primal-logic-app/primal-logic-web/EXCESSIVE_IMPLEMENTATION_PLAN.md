# 「一気に余計なくらい大量に実装」適用箇所

## 1. 食品データベース（最優先）

### 現状
- 基本的な食品のみ
- 栄養素データが限定的

### 大量実装案
- **1000種類以上の食品を追加**
  - 各部位の詳細バリエーション（リブアイ、サーロイン、フィレ、チャック等）
  - 調理方法別（生、焼く、煮る、ロースト、スモーク等）
  - 部位別の詳細（骨付き、骨なし、皮付き、皮なし等）
  - 魚類の詳細（サーモン、マグロ、サバ、イワシ等、各部位）
  - 卵の種類（鶏卵、うずら卵、ダック卵等）
  - 内臓の詳細（レバー、心臓、腎臓、脳、舌等）
  - 骨髄、軟骨、皮等の詳細
- **栄養素データの完全性**
  - 全栄養素を網羅（ビタミンB群全種類、ミネラル全種類、脂肪酸詳細等）
  - 調理による栄養素変化を考慮
  - 部位による栄養素の違いを詳細に

### ON/OFF機能
- 「詳細食品データベース表示」（デフォルトOFF）
- 「調理方法別栄養素表示」（デフォルトOFF）
- 「部位別詳細表示」（デフォルトOFF）

---

## 2. 栄養素ゲージ（高優先度）

### 現状
- 主要栄養素のみ表示
- 「全て表示」ボタンで切り替え

### 大量実装案
- **全栄養素のゲージを追加**
  - ビタミンB群全種類（B1, B2, B3, B5, B6, B7, B9, B12）
  - ビタミンE, K2（MK-4, MK-7）
  - ミネラル全種類（亜鉛、銅、セレン、マンガン、クロム、モリブデン等）
  - 脂肪酸詳細（飽和脂肪酸、オメガ3, オメガ6, オメガ9等）
  - アミノ酸プロファイル（必須アミノ酸9種類）
  - コレステロール、コリン、タウリン等
- **栄養素の詳細分析**
  - 栄養素間の相互作用表示
  - 吸収率の可視化
  - バイオアベイラビリティの表示

### ON/OFF機能
- 「詳細栄養素表示」（デフォルトOFF）
- 「栄養素相互作用表示」（デフォルトOFF）
- 「バイオアベイラビリティ表示」（デフォルトOFF）
