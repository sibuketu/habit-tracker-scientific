# オンボーディング画面の改善

## 実装した改善

### 1. 言語設定を最初のステップに追加
- オンボーディングの最初のステップで言語選択を表示
- 5つの言語（日本語、英語、フランス語、ドイツ語、中国語）から選択可能
- 選択した言語は即座に反映される

### 2. 「戻る」ボタンの追加
- 最初のステップ以外では「戻る」ボタンを表示
- 前のステップに戻ることができる

### 3. AI Speed Dialの非表示
- オンボーディング中はAI Speed Dialボタンを非表示
- オンボーディング完了後に表示される
