# オンボーディングで説明が必要なAI機能

## 8つの機能（ナンバリング）

### 1. AIチャット（最重要）
- **機能**: 操作方法や機能について何でも質問できる
- **使い方**: 右下のAIボタンをタップ → 「💬 AIチャット」を開く
- **説明内容**: 
  - 「使い方がわからない時は、右下のAIボタンをタップして質問してください」
  - 「操作方法や機能について、AIが詳しく説明します」
  - 「カーニボアダイエットの専門知識も質問できます」

### 2. 音声入力（自動記録）
- **機能**: 話すだけで食品記録や日次入力が自動で完了
- **使い方**: AIチャット内で音声入力ボタンをタップ → 話すだけ
- **説明内容**:
  - 「入力が面倒なら、音声で話しかけるだけで記録できます」
  - 「例：『サーモン200g食べた』と話すだけで自動記録」
  - 「例：『今日の睡眠は95点、8時間、太陽光は30分、排泄は正常』と話すだけで日次入力も完了」
  - 「画面を見ながら話すだけで、全ての機能を操作できます」

### 3. 写真から食品追加
- **機能**: 食べた食品の写真を撮るだけで、AIが自動で栄養素を計算
- **使い方**: AI Speed Dial → 「📷 写真から追加」をタップ → 写真を撮る
- **説明内容**:
  - 「食べた食品の写真を撮るだけで、AIが自動で栄養素を計算します」
  - 「複数の食品が写っていても、AIが自動で識別します」

### 4. 栄養素不足の警告と提案
- **機能**: 栄養素が不足している時、AIが自動で警告し、推奨食品を提案
- **使い方**: 自動で警告される（AIチャットでも質問可能）
- **説明内容**:
  - 「栄養素が不足している時、AIが自動で警告し、推奨食品を提案します」

### 5. 症状に応じた栄養素目標値の自動調整
- **機能**: 症状を話すと、AIが自動で栄養素目標値を調整
- **使い方**: AIチャットで「頭痛がする」などと質問
- **説明内容**:
  - 「症状を話すと、AIが自動で栄養素目標値を調整します」
  - 「例：『頭痛がする』→ナトリウム目標値を自動調整」

### 6. 移行期間のサポート
- **機能**: カーニボア移行期間中、AIが症状を検出して自動で対処法を提案
- **使い方**: 移行期間中、自動でサポートされる（AIチャットでも質問可能）
- **説明内容**:
  - 「カーニボア移行期間中、AIが症状を検出して自動で対処法を提案します」

### 7. リカバリープロトコル生成
- **機能**: 糖質を摂取してしまった時、AIが最適なリカバリープロトコルを提案
- **使い方**: 糖質を摂取した時、自動で提案される（初回体験時に説明）
- **説明内容**:
  - 「糖質を摂取してしまった時、AIが最適なリカバリープロトコルを提案します」
  - **注意**: オンボーディングでは説明しない。ユーザーが初めて体験する時に説明する

## オンボーディングで説明すべき優先順位

1. **AIチャット**（必須・最重要）
2. **音声入力**（必須・入力の手間を大幅に削減）
3. **写真から食品追加**（推奨・便利機能）
4. **その他の機能**（オプション・必要に応じて説明）
5. **リカバリープロトコル**（オンボーディングでは説明しない。初回体験時に説明）

## 説明方法

- **最小限**: AIチャットと音声入力のみ
- **標準**: AIチャット、音声入力、写真から食品追加
- **詳細**: 上記全ての機能（リカバリープロトコル除く）
