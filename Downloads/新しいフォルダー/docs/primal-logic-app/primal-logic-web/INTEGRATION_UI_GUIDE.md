# 連携機能UIガイド

## 天気情報取得機能（OpenWeatherMap API）

### 🌤️ボタンの場所

**InputScreen（今日の記録画面）**の「太陽光暴露（ソーラーチャージ）」セクションにあります。

#### アクセス方法
1. ホーム画面（HomeScreen）を開く
2. 「📝 今日の記録」ボタンをクリック → InputScreenが開く
3. 画面をスクロールして「太陽光暴露（ソーラーチャージ）」セクションを見つける
4. Sunny/Cloudyボタンの右側に**🌤️ボタン**があります

### 使用方法

1. **🌤️ボタンをクリック**
   - 現在地の天気情報を自動取得します
   - 位置情報の許可を求められる場合があります

2. **自動反映**
   - 天気情報が取得されると、自動的に表示されます
   - Sunny/Cloudyボタンが自動的に設定されます
   - ビタミンD合成計算に天気情報が反映されます

3. **表示される情報**
   - 都市名
   - 天気状態（☀️ 晴れ / ⛅ 曇りがち / ☁️ 曇り / 🌧️ 雨 / ❄️ 雪）
   - 気温（℃）
   - UV指数

### 天気情報が表示されない場合

- APIキーが設定されていない（`.env`ファイルに`VITE_OPENWEATHER_API_KEY`を追加）
- 位置情報の許可が拒否された
- ネットワークエラー

これらの場合は、手動でSunny/Cloudyボタンをクリックして設定できます。

## Google Fit連携

### 🔄 Google Fitから取得ボタンの場所

**HealthDeviceScreen（健康デバイス連携画面）**にあります。

#### アクセス方法
1. ホーム画面 → 「⚙️ 設定」を開く
2. 「健康デバイス連携」を開く → HealthDeviceScreenが開く
3. 画面上部に**🔄 Google Fitから取得**ボタンがあります

### 使用方法

1. **🔄 Google Fitから取得ボタンをクリック**
   - Google Fitからデータを取得します
   - OAuth 2.0認証が必要（現在未実装）

2. **自動反映**
   - 歩数、心拍数、活動時間、消費カロリーが自動的にフォームに反映されます

### 現在の実装状況

- ⚠️ OAuth 2.0認証は未実装
- 基本的な構造とUIは実装済み
- 将来的にOAuth 2.0認証を実装予定

詳細は`INTEGRATION_IMPLEMENTATION_STATUS.md`を参照してください。
