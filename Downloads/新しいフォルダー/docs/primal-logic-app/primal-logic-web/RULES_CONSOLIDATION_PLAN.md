# Rules統合・整理計画

## 現状の問題
- Rulesが多すぎて、いらなそうなものが混在している
- 重複や矛盾がある可能性
- 優先度が不明確

## 整理方針

### 1. 重複の統合
- 「提案の扱い方」と「提案を守るためのチェックリスト」は統合可能
- 「Web検索の優先」と「情報の正確性の検証」は統合可能

### 2. 優先度の明確化
- 【最重要】→ 絶対に守る
- 【重要】→ 基本的に守る
- その他 → 参考程度

### 3. 削除候補
- 具体的すぎる例（他のルールでカバーできる）
- 重複している内容
- 実装完了済みで不要になった内容

## 次のステップ
1. Rules全体を読み直し
2. 重複・矛盾を特定
3. 統合・削除を実行
4. 優先度順に再配置
