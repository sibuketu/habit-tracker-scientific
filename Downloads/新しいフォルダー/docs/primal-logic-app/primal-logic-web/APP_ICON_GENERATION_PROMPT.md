# Carnivore専用健康管理アプリ アプリアイコン生成プロンプト

## 使用ツール
- GeminiNanobanana（画像生成）

## プロンプト

```
Carnivore専用健康管理アプリのアプリアイコンを大量生成してください。

【必須要件】
- 肉の写真を必ず含める（ステーキ、ハンバーグ、焼き肉など）
- 健康的で魅力的なデザイン
- アプリアイコンとして使用可能なサイズ（1024x1024ピクセル推奨）
- 複数のバリエーションを生成

【デザイン要素】
- 肉の写真（必須）
- 健康管理アプリらしいシンプルで分かりやすいデザイン
- カーニボアダイエットを連想させる要素
- モダンで洗練された見た目

【バリエーション例】
- ステーキを中心にしたデザイン
- ハンバーグを中心にしたデザイン
- 焼き肉を中心にしたデザイン
- 肉と栄養素を組み合わせたデザイン
- 肉と健康をテーマにしたデザイン

大量に生成してください。
```
