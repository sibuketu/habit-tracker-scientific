# SNS動画・アプリのスタンス方針

## 検討事項

### 「賢いあなたなら分かるでしょう」的なスタンスについて

**提案1: 控えめに使用**
- 時々使うのはOKだが、多用しない
- 使う場面：技術的な説明や複雑な概念を説明した後、「賢いあなたなら理解できるでしょう」という形で締める
- 避ける場面：基本的な説明や初心者向けの内容

**提案2: 別の表現に置き換える**
- 「頭が柔らかいあなたなら」→ 気持ち悪い（ユーザー指摘）
- 代替案：
  - 「理解できるでしょう」
  - 「分かりますよね」
  - 「説明不要かもしれませんが」
  - 「念のため説明しますが」
  - 「ご存知かもしれませんが」

**提案3: 使わない**
- 全ての説明を丁寧に行う
- ユーザーの理解度を前提としない
- 初心者にも分かりやすい説明を心がける

## 推奨方針

**「控えめに使用 + 別の表現に置き換える」を推奨**

- 基本的には丁寧な説明を心がける
- 技術的な説明や複雑な概念の後、軽く「理解できるでしょう」程度に留める
- 「賢いあなたなら」という表現は避け、「理解できるでしょう」「分かりますよね」などの自然な表現を使用
- 初心者にも分かりやすい説明を基本とする

## 実装方針

- アプリ内の説明文：基本的に丁寧な説明を心がける
- SNS動画：技術的な説明の後、軽く「理解できるでしょう」程度に留める
- 全ての説明を丁寧に行うことを基本とする
