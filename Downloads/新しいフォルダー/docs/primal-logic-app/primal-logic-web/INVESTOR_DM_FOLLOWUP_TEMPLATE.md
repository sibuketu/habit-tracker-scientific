# 過去に連絡した投資家へのフォローアップメッセージテンプレート

## 使用場面
以前習慣化アプリで連絡した投資家に、CarnivoreGuidアプリについて再度連絡する場合

## テンプレート

```
[投資家の名前]さん、お久しぶりです。[以前連絡した日付やきっかけを記載する箇所]。

以前は習慣化アプリについてご相談させていただきましたが、今回は新しいプロジェクト「CarnivoreGuid」についてご連絡させていただきます。

【CarnivoreGuidとは】
カーニボアダイエット管理アプリです。このアプリは単なる「記録ツール」ではなく、「カーニボアという急成長市場における、唯一の論理的プラットフォーム」を目指しています。

【主な機能】
- 栄養素追跡（タンパク質、脂質、ビタミン、ミネラルなど）: ユーザープロファイル（100項目以上の質問）に基づいて動的に栄養素目標値を調整
- AIチャット: Carnivore専門家（Dr. Ken Berry、Dr. Shawn Baker、Dr. Paul Saladino、Dr. Anthony Chaffee、Dr. Bart Kayなど）の情報を統合。情報源がCarnivore医者であり、嘘情報を事前にはじいている
- 写真解析: 食べたものを写真で記録し、AI Conciergeインタビューで詳細を補完（塩、脂質のトリミング、調理方法など）
- P:F Ratio（タンパク質:脂質比率）: ホーム画面の最上部にゲージとして常時表示

【技術スタック】
- フロントエンド: React (Vite) + TypeScript
- AI: Gemini API（AIチャット、画像解析）
- ストレージ: localStorage（Privacy First）
- デプロイ: Netlify
```

## 使い方
1. `[投資家の名前]`を実際の名前に置き換える
2. `[以前連絡した日付やきっかけを記載する箇所]`に、以前の連絡内容を簡潔に記載（例：「以前は習慣化アプリについてご相談させていただきました」）
3. `[以前の会話で触れた内容や、その投資家のアドバイスを参考にした点を記載する箇所]`に、以前の会話の要点を記載（会話内容を全部貼るのではなく、要点をまとめる）

## 会話内容を貼る場合の注意点
- 会話内容を全部貼るのは長すぎる可能性がある
- 要点をまとめて、その投資家のアドバイスや意見を参考にした点を簡潔に記載する
- 例：「以前ご指摘いただいた[具体的なアドバイス]を参考に、今回は[どう活かしたか]を意識して開発しました」
