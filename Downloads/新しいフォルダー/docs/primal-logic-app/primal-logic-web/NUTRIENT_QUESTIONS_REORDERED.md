# 栄養素に関する質問項目 - 変動量順（再整理版）

## 優先度の基準（再定義）
1. **変動量の大きさ**: 栄養素目標値への影響度が高い順（数値で変動する量が大きい順）
2. **カバー範囲**: より多くの栄養素に影響する順
3. **実用性**: 実際に測定・把握しやすい順

## 【最優先】変動量が大きい項目（栄養素目標値に直接影響、±20%以上）

1. **性別** - 鉄分（8mg→18mg、+125%）、タンパク質（+20%）
2. **活動量** - タンパク質（+20%）、脂質（+20%）、マグネシウム（+100mg、+17%）

4. **授乳中**（女性） - タンパク質（+30%）、マグネシウム（+17%）
5. **年齢（50歳以上）** - ビタミンD（+50%）、タンパク質（+10%）
6. **運動強度（激しい）** - タンパク質（+30%）、脂質（+27%）、マグネシウム（+25%）
