# 投資家探し・連絡ガイド

## ゴール: 世界一のCarnivoreアプリを作る

このガイドは、ゴール達成に貢献できる投資家（パートナー）を見つけるためのものです。

---

## 【最重要】投資家とは何をするのか？

### 投資家との関係性
投資家は「お金をくれるだけ」ではありません。ゴール達成のための**パートナー**です。

**投資家から得られるもの:**
1. **資金**: アプリ開発、マーケティング、人材採用などに必要な資金
2. **戦略的アドバイス**: ビジネス戦略、市場分析、競合分析などのアドバイス
3. **ネットワーク**: 他の起業家、専門家、パートナー企業とのつながり
4. **メンタリング**: 起業家としての成長をサポートするメンタリング
5. **信頼性**: 投資家がつくことで、他の投資家やパートナーからの信頼が得られる

**投資家に提供するもの:**
1. **株式**: プロジェクトの一部を提供（例: 10-15%。会社設立前の場合は、会社設立後の株式を約束する）
2. **成長**: アプリが成長することで、投資家の投資がリターンを生む
3. **情報**: 定期的な進捗報告、財務状況の共有

**良い投資家の特徴:**
- 単なる資金提供者ではなく、**戦略的パートナー**として関与する
- 短期的な利益ではなく、**長期的な成長**を重視する
- カーニボアダイエットや健康テックへの**理解**がある、または理解しようとする姿勢
- **経験豊富**で、健康/SaaS分野での投資実績がある

**避けるべき投資家:**
- 「お金をくれるだけ」で、その後は「頑張ってね」と放置する投資家
- 条件交渉にばかりこだわり、パートナーシップを重視しない投資家
- カーニボアダイエットや健康テックへの理解がない投資家
- 短期的な利益のみを追求する投資家

### 投資家選定の基本方針
- **条件の良さ（株式割合、評価額）よりも、ゴール達成に貢献できるか**を最優先に判断する
- 短期的な条件交渉よりも、長期的なパートナーシップを重視する
- 10%や15%の株式を気にするより、良い人に出資してもらう方が重要

### 2. Smartround / 日本の投資家マッチングプラットフォーム（推奨）

**推奨順位**: A（StartupList）→ B（ANGELMatching）→ C（スタートアップマッチングスクエア）→ D（Smartround）→ E（SmartPitch）

#### A. StartupList（スタートアップリスト）【最推奨】
- **URL**: Web検索で確認（例: startuplist.jp）
- **特徴**: 国内最大級の起業家・投資家マッチングサービス
- **メリット**: 定期的なピッチイベント、細やかなサポート、投資家が積極的に見ている
- **使い方**: 
  1. Web検索で「StartupList」を検索して公式サイトを確認
  2. アカウント作成
     - **Googleアカウントでの登録**: 自分から見えるメールアドレス（例：example@gmail.com）で登録する。プラットフォーム上で表示されるプロフィール名（相手から見える情報）は、後で設定画面で変更可能な場合が多い。登録時はメールアドレスを使用し、プロフィール情報は後で設定する。
  3. **会社情報の登録**（必須項目）:
     - **会社設立有無**: 「会社設立前」を選択
     - **代表(CEO)の氏名**: 須佐見 隼人
     - **電話番号**: 自分の電話番号を入力
     - **事業を一言で**（100文字以内）: 例：「カーニボアダイエット管理アプリ「CarnivoreGuid」の開発・運営。AIチャット、栄養素追跡、写真解析などの機能を提供し、カーニボアという急成長市場における唯一の論理的プラットフォームを目指す。」
     - **コーポレートサイトの有無**: 「ない」を選択
       - **コーポレートサイトとは**: 会社の公式ウェブサイト（会社の情報、事業内容、会社概要などを掲載するサイト）のこと。会社を設立していない場合は「ない」を選択する。
     - **コーポレートサイトURL**: 「ない」を選択した場合は入力不要
  4. **任意項目**（審査がスムーズになるため、可能な限り入力）:
     - **事業概要**（500文字以内）: 以下のコードブロックからコピペして使用
     - **代表(CEO)の経歴**（500文字以内）: 以下のコードブロックからコピペして使用
     - **投資家向け資料リンク**: スライド資料のURL（https://docs.google.com/presentation/d/1DFIYDSsZZsjssx_vagENn9MLQrmbatkO/edit?usp=sharing&ouid=104172839611674735492&rtpof=true&sd=true）を入力
     - **紹介者**: 該当する場合は入力、なければ空欄
  5. 投資家を検索・連絡

### 3. LinkedIn / 投資家の公式サイト
**優先度**: 中

**LinkedIn:**
- プロフェッショナルなネットワーク
- 投資家の経歴や実績を確認できる
- DM送信が可能（制限が少ない）
- **使い方**: LinkedInアカウント作成 → 投資家を検索 → DM送信

**投資家の公式サイト・ブログ:**
- 連絡フォームやメールアドレスが公開されている場合がある
- 投資家の価値観や投資方針を詳しく確認できる
- **使い方**: 各投資家の公式サイトを確認 → 連絡フォームやメールアドレスを探す → メッセージを送信

### 4. ピッチイベント・スタートアップコンテスト
**優先度**: 低（時間とコストがかかる）

**メリット:**
- 直接会って話せる
- 複数の投資家に一度にアピールできる
- フィードバックが得られる

**デメリット:**
- 参加費用がかかる場合がある
- 準備に時間がかかる

**使い方:**
1. Web検索で「ピッチイベント スタートアップ 投資家 2025」などを検索
2. イベント情報を確認（日時、場所、参加条件、費用など）
3. 参加申し込み
4. ピッチ資料を準備（スライド資料を活用）
5. イベントに参加

## 投資家へのメッセージ作成ガイド

### 基本方針
- **条件交渉ではなく、ゴール共有とパートナーシップの提案**に焦点を当てる
- カーニボアダイエットとは何かを明確に説明する
- アプリの価値提案（単なる記録ツールではなく、唯一の論理的プラットフォーム）を伝える
- 投資家ごとにパーソナライズする

### メッセージの構成
1. **挨拶**: 簡潔に自己紹介（龍谷大学一年の須佐見 隼人）
2. **カーニボアダイエットとは**: 明確に説明（投資家が知らない可能性がある）
3. **アプリの特徴**: 主な機能を簡潔に紹介
4. **技術スタック**: 実装済みであることを示す
5. **デモURL**: 実際に確認できるようにする
6. **パーソナライズ**: 投資家の実績や興味に触れる
7. **投資依頼**: 明確に伝える（1500万円）

## 投資家連絡の具体的な手順

### ステップ1: プラットフォーム選定と登録
1. **優先度A（X DM）**: 既に実施済み ✅
2. **優先度B（マッチングプラットフォーム）**: 
   - StartupListから始める（最推奨）
   - Web検索で「StartupList」を検索して公式サイトを確認
   - アカウント作成
   - プロジェクト情報を登録（CarnivoreGuidの詳細、デモURL、スライド資料など）

### ステップ2: 投資家検索と連絡
1. **投資家を検索**: 
   - プラットフォーム内で投資家を検索
   - 健康/SaaS分野、エンジェル投資家、スタートアップ支援に興味がある投資家を探す
2. **投資家のプロフィールを確認**: 
   - 投資実績、投資分野、投資スタイルを確認
   - カーニボアダイエットや健康テックへの理解があるか確認
3. **メッセージを作成**: 
   - `INVESTOR_DM_COMPLETE.md`を参考に、投資家ごとにパーソナライズ
   - カーニボアダイエットとは何かを明確に説明
   - アプリの価値提案を伝える
   - デモURLとスライド資料のURLを記載
4. **メッセージを送信**: 
   - 1日数件に抑える
   - 返事を待つ

### ステップ3: フォローアップ
1. **返事が来た場合**: 
   - 詳細を共有
   - 必要に応じて、他のプラットフォーム（Smartroundなど）に移行
2. **返事が来ない場合**: 
   - 1週間後に再送信を検討
   - または、他のプラットフォーム（ANGELMatching、スタートアップマッチングスクエアなど）を試す

### 同時進行の原則
- アプリ開発、動画生成、SNS投稿と同時進行で進める
- 明示的に「絞る」と言われない限り、全てのタスクを並行して実行する
