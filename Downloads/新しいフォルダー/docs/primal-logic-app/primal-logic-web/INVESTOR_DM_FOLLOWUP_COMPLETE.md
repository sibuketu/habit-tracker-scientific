# 過去に連絡した投資家へのフォローアップメッセージ（完全版）

## ゴール: 世界一のCarnivoreアプリを作る

以前習慣化アプリで連絡した投資家に、CarnivoreGuidアプリについて再度連絡するためのメッセージです。

---

## 1. Gazelle Capital 近藤氏へのフォローアップ

```
近藤さん、お久しぶりです。以前は習慣化アプリについてご相談させていただいた、龍谷大学一年の須佐見 隼人です。

今回は新しいプロジェクト「CarnivoreGuid」についてご連絡させていただきます。

【カーニボアダイエットとは】
肉、魚、卵、内臓などの動物性食品のみを摂取し、植物性食品を完全に除去する食事法です。植物毒を避け、血糖値の安定、満腹感の持続、腸の健康改善を目的としています。世界的に急成長している健康ムーブメントであり、科学的根拠に基づいた実践者が増えています。

【CarnivoreGuidとは】
カーニボアダイエット管理アプリです。このアプリは単なる「記録ツール」ではなく、「カーニボアという急成長市場における、唯一の論理的プラットフォーム」を目指しています。

【主な機能】
- 栄養素追跡（タンパク質、脂質、ビタミン、ミネラルなど）: ユーザープロファイル（100項目以上の質問）に基づいて動的に栄養素目標値を調整
- AIチャット: Carnivore専門家（Dr. Ken Berry、Dr. Shawn Baker、Dr. Paul Saladino、Dr. Anthony Chaffee、Dr. Bart Kayなど）の情報を統合。情報源がCarnivore医者であり、嘘情報を事前にはじいている
- 写真解析: 食べたものを写真で記録し、AI Conciergeインタビューで詳細を補完（塩、脂質のトリミング、調理方法など）
- P:F Ratio（タンパク質:脂質比率）: ホーム画面の最上部にゲージとして常時表示

【技術スタック】
- フロントエンド: React (Vite) + TypeScript
- AI: Gemini API（AIチャット、画像解析）
- ストレージ: localStorage（Privacy First）
- デプロイ: Netlify

【市場機会】
カーニボアダイエットは世界的なムーブメントであり、健康意識の高い層がターゲットです。既存の栄養管理アプリは「カロリー計算」中心ですが、CarnivoreGuidは「カーニボア専門」で差別化しています。

【現在の状況】
主要機能を実装済みで、デモURLも公開中です。
デモURL: https://scintillating-dodol-e8fcf2.netlify.app/
スライド資料: https://docs.google.com/presentation/d/1DFIYDSsZZsjssx_vagENn9MLQrmbatkO/edit?usp=sharing&ouid=104172839611674735492&rtpof=true&sd=true

【以前のアドバイスを活かした点】
以前ご指摘いただいた「誰にどう使われるか（ヒアリング・属性分析）」の視点を、今回のCarnivoreGuidでも重視しています。カーニボアダイエット実践者の属性（年齢、性別、健康状態、実践期間など）を100項目以上の質問で把握し、それに基づいて栄養素目標値を動的に調整する設計にしています。また、「広告の外部委託をやめるべき（今はそのフェーズではない）」というご指摘を踏まえ、プロダクト完成とユーザー検証を優先しています。

1500万円の資金調達をご検討いただけますでしょうか？

よろしくお願いいたします。
須佐見 隼人
```
