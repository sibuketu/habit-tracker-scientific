## 動作確認方法

### 1. 開発サーバーが起動しているか確認

ブラウザで以下にアクセス:
```
http://localhost:5173
```

### 2. ブラウザの開発者ツールで確認

1. **ブラウザでアプリを開く**（F12で開発者ツールを開く）
2. **Consoleタブ**を開く
3. 以下を実行：

```javascript
console.log(import.meta.env.VITE_GEMINI_API_KEY);
```

**正しいAPIキーが表示されればOK。**
`undefined`が表示される場合は、開発サーバーを再起動してください。

### 3. AIチャット機能をテスト

1. アプリで「✨」ボタンをクリック
2. 「💬 Chat」をクリック
3. 「足痛い」などと入力して送信
4. エラーメッセージが表示されるか確認

### 4. エラーが出る場合の対処

**「Gemini APIキーが設定されていません」と表示される場合:**

1. 開発サーバーを完全に停止（Ctrl+C）
2. 再起動: `npm run dev`
3. ブラウザをハードリロード（Ctrl + Shift + R）

**「API呼び出しエラー」と表示される場合:**

1. APIキーが正しいか確認（Google AI Studioで確認）
2. APIキーにクレジットが残っているか確認
3. ネットワーク接続を確認
