# Netlifyダッシュボードでのコミットハッシュ確認ガイド

**目的**: Netlifyのデプロイログから、3日前にデプロイされたコミットハッシュを確認する

---

## 手順

### Step 1: Netlifyにログイン

1. ブラウザで https://app.netlify.com にアクセス
2. GitHub/Google/GitLab/Bitbucketでログイン

### Step 2: サイトを選択

1. ダッシュボードでサイト一覧が表示されます
2. **「strong-travesseiro-0a6a1c」** をクリック

### Step 3: Deploysタブを開く

1. サイトの詳細ページが開きます
2. 上部のタブから **「Deploys」** をクリック

### Step 4: 3日前のデプロイを選択

1. デプロイ履歴が時系列で表示されます
2. **3日前（2026-01-22頃）** のデプロイをクリック

### Step 5: ビルドログを確認

1. デプロイの詳細ページが開きます
2. **「Deploy log」** または **「Build log」** をクリック
3. ログの最初の方に以下のような情報が表示されます：

```
Commit: 2a87694e4ae864125827b2c17cb54e401e54b17e
Branch: main
Author: sibuketu <sibuketu@gmail.com>
```

### Step 6: コミットハッシュをコピー

- **「Commit:」** の後に表示されているハッシュ（例: `2a87694e4ae864125827b2c17cb54e401e54b17e`）をコピー

---

## 代替方法: デプロイの詳細ページから確認

デプロイの詳細ページには、コミット情報が直接表示されている場合もあります：

1. デプロイの詳細ページを開く
2. 右上または上部に **「Commit」** というリンクや情報が表示されている
3. そこをクリックすると、GitHubのコミットページに移動します
4. URLからコミットハッシュを確認できます

例: `https://github.com/sibuketu/CarnivoreGuid/commit/2a87694e4ae864125827b2c17cb54e401e54b17e`

---

## コミットハッシュが見つからない場合

もしコミットハッシュが見つからない場合は、以下を確認してください：

1. **手動デプロイ**: Netlifyのダッシュボードから手動でアップロードした場合、コミット情報が表示されない可能性があります
2. **別のリポジトリ**: Netlifyが別のリポジトリからデプロイしている可能性があります
3. **ビルド設定**: Netlifyのビルド設定で、リポジトリの接続方法を確認してください

---

## 次のステップ

コミットハッシュを取得したら、以下のコマンドでそのコミットからソースコードを取得できます：

```powershell
# 特定のコミットからファイルを取得
git checkout <commit-hash> -- src/screens/HomeScreen.tsx
git checkout <commit-hash> -- src/screens/HistoryScreen.tsx
# ... 他のファイルも同様に
```
