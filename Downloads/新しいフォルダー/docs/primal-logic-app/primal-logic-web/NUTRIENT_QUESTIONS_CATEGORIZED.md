# 栄養素に関する質問項目 - 検査不要/必要で分類 + 変動量順

## 分類基準

## UI設計の推奨

### セクション1: 基本情報（検査不要、最優先）
- 性別、年齢、活動量、ストレスレベル

### セクション2: ライフスタイル（検査不要、高優先度）
- 睡眠時間、睡眠の質、運動強度、運動頻度、日光暴露頻度、水分摂取量

### セクション3: 健康状態（検査不要、中優先度）
- 消化器系、炎症レベル、メンタルヘルス、症状（頭痛、片頭痛、ブレインフォグ等）

### セクション4: 疾患・症状（検査不要、中優先度）
- 各種疾患（IBS、クローン病、糖尿病等）

### セクション5: 女性特有（検査不要、高〜中優先度）
- 妊娠、授乳、閉経、PMS、PCOS等

### セクション6: サプリメント・薬物（検査不要、低優先度）
- サプリメント摂取状況、薬物使用リスト

### セクション7: 体組成（検査必要、オプション）
- 体脂肪率、筋肉量

### セクション8: 血液検査値（検査必要、オプション）
- 各種血液検査値（データがある場合のみ）
