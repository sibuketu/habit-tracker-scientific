# 「植物性」と「避けるべき」の違いについて

## 現在の実装

CustomFoodScreenでは、食品タイプとして以下の3つを選択できます：

1. **動物性（animal）**: 動物由来の食品（肉、魚、卵など）
2. **植物性（plant）**: 植物由来の食品（野菜、果物、穀物など）
3. **避けるべき（trash）**: カーニボアダイエットで避けるべき食品全般

## 違いの説明

### 「植物性」と「避けるべき」の違い

- **「植物性」**: 
  - 植物由来の食品（野菜、果物、穀物など）
  - カーニボアダイエットでは基本的に避けるが、栄養素データとして記録する場合に使用
  - 例: 野菜を食べた場合に記録する、栄養素データとして保存する

- **「避けるべき」**: 
  - カーニボアダイエットで避けるべき食品全般（植物性食品を含む）
  - より明確に「避けるべき」ことを示す
  - 例: 誤って食べてしまった場合、明確に避けるべき食品として記録する

## 問題点

実際には、「植物性」と「避けるべき」は重複している可能性があります。

- カーニボアダイエットでは、植物性食品は基本的に避けるべき
- 「植物性」を選択する場合と「避けるべき」を選択する場合の違いが曖昧
- ユーザーにとって混乱を招く可能性がある

## 提案

### オプション1: 「植物性」と「避けるべき」を統合
- 「植物性」を削除し、「避けるべき」のみにする
- 植物性食品を記録する場合も「避けるべき」を選択

### オプション2: 説明を追加
- 「植物性」と「避けるべき」の違いを明確に説明する
- ツールチップや説明文を追加

### オプション3: 現状維持
- 実装したままにする
- ユーザーが使い分けできる

## 推奨

**オプション1を推奨**します。理由：
- カーニボアダイエットでは、植物性食品は基本的に避けるべき
- 選択肢が少ない方が分かりやすい
- 「植物性」と「避けるべき」の違いは実用的ではない

ただし、将来的に「植物性食品を食べた場合の栄養素データを記録する」という用途がある場合は、現状維持も検討可能です。

## 決定が必要

ユーザーの判断をお願いします：
1. 「植物性」と「避けるべき」を統合する（「植物性」を削除）
2. 現状維持（実装したまま）
3. その他の提案
