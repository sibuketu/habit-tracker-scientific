Carnivore専門家の皆様へ

このアプリは、カーニボアダイエット実践者のための世界最高峰の健康管理アプリを目指しています。以下の機能をご覧ください：

1. 栄養素追跡（世界最高水準）

4ゾーングラデーションゲージ
- 不足ゾーン（0-50%）: 赤色で表示。栄養素が目標値の50%未満の場合
- バッファーゾーン（50-100%）: オレンジ色で表示。栄養素が目標値の50-100%の場合
- 最適ゾーン（100-120%）: 緑色で表示。栄養素が目標値の100-120%の場合（最適範囲）
- 過剰ゾーン（120%以上）: 紫色で表示。栄養素が目標値の120%を超える場合
- 特徴: 単なるバーではなく、グラデーションで視覚的にわかりやすく表示。各栄養素の状態を一目で把握可能

動的目標値システム（100項目以上のプロファイル設定）
- 個人最適化: 性別、年齢、活動レベル、妊娠/授乳状態、ストレスレベル、睡眠時間、運動強度・頻度、甲状腺機能、太陽光暴露頻度、消化器系の問題、炎症レベル、メンタルヘルス状態、サプリメント摂取状況、アルコール・カフェイン摂取量、カーニボア開始日（移行期間判定）、体組成、代謝ストレス指標などを考慮
- カーニボア基準: Dr. Ken Berry、Dr. Shawn Baker等の専門家の推奨値に基づく
- リアルタイム計算: 食品追加時に自動的に目標値を再計算

P:F比率表示
- タンパク質と脂質の比率を視覚化
- カーニボアダイエットの重要な指標をリアルタイムで表示

主要栄養素の追跡
- マクロ栄養素: タンパク質、脂質、炭水化物、正味炭水化物、食物繊維
- ミネラル: ナトリウム、マグネシウム、カリウム、亜鉛、鉄、カルシウム、リン、セレン、銅、マンガン、ヨウ素、クロム、モリブデン、フッ素、塩素、ホウ素、ニッケル、ケイ素、バナジウム
- 脂溶性ビタミン: ビタミンA、ビタミンD、ビタミンK、ビタミンK2、ビタミンE
- 水溶性ビタミン: ビタミンC、ビタミンB1、B2、B3、B5、B6、B7（ビオチン）、B9（葉酸）、B12
- その他: コリン、タウリン、オメガ3、オメガ6、オメガ比率
- 比率: カルシウム:リン比率、グリシン:メチオニン比率、P:F比率
- 抗栄養素: フィチン酸、ポリフェノール、フラボノイド、アントシアニン、シュウ酸、レクチン、サポニン、ゴイトロゲン、タンニン、トリプシン阻害物質、プロテアーゼ阻害物質、シアノグリコシド、ソラニン、チャコニン

2. AI機能（専門家モード）

AIチャット
- 情報源: Carnivore専門家（Dr. Ken Berry、Dr. Shawn Baker、Dr. Paul Saladino、Dr. Anthony Chaffee、Dr. Bart Kay、Dr. Paul Mason）の主張に基づく
- 特徴: 普通のAIとは異なり、情報源がCarnivore医者であり、嘘情報を事前にはじいている
- 論証スタイル: 4ステップ論証（検証 → 脱構築 → 再構築 → 引用）で、誤解や批判に対して体系的に対応
- Detective AI: 日記と食生活のデータを分析して、症状の原因を特定（「なぜ調子が悪いの？」などの質問に対応）
- 実践的アドバイス: 症状や問題に対して、具体的な対処法を提示（例：「頭痛がする」→「ナトリウムを+2000mg増やしてください」）

写真解析
- 食べたものを写真で記録
- AIによる栄養素の自動検出（Gemini Vision API使用）

AI Concierge
- 詳細な栄養素情報の自動補完
- 食品名から栄養素を推測

3. 食品データベース

食品選択UI（タブ切り替え方式）
- 動物選択: タブ切り替えで牛・豚・鶏・魚・卵・乳製品・脂質・植物性食品を選択
- 部位選択: 文字選択でリブアイ、ブリスケット、サーロイン、肝臓、内臓などを選択
- リアルタイム栄養素計算: 選択中の食品の栄養素をプレビュー表示
- 量調整: スライダーまたは数値入力で量を調整
- 栄養素内訳表示: タンパク質、脂質、ミネラル、ビタミンなどの詳細表示
- 注: 現在は解剖図タップではなく、タブ切り替えと文字選択によるUIを採用

カスタム食品登録
- ユーザーが独自の食品を登録可能
- 栄養素情報を手動で設定可能

履歴活用（History Copy機能）
- 「いつもの（My Foods）」タブで過去の入力履歴からワンタップ入力
- よく使う食品を素早く選択可能

4. 違反検知とリカバリー

違反検知
- 植物性食品や加工食品の摂取を自動検知
- 抗栄養素の摂取量を追跡
- 違反タイプの自動分類

Recovery Protocol（自動生成）
- 違反検出時に自動的にリカバリープロトコルを生成
- AIを活用したパーソナライズされたリカバリー提案
- 断食時間の自動計算
- 推奨食品の自動提案
- 「明日のログに追加」機能で翌日のログに自動追加

Recovery Quest
- Todoリスト形式でリカバリープロトコルを実行
- タイマー機能、食品追加、プロトコル設定などのアクション

5. Logic Armor（理論武装機能）

KnowledgeScreen
- カーニボアに対する誤解（Myth）と真実（Truth）をカード形式で表示
- カテゴリフィルター: 消化、心臓、長期健康、栄養などでフィルタリング
- フリップカードUI: タップでMythからTruthに切り替え

Argument Cards
- 栄養素に関する科学的根拠を表示
- 3-tier表示（Glance → Explanation → Source）
- 栄養素をタップするとArgument Cardが表示される
- 反論に備えた知識の整理

6. 習慣トラッカー（ストリーク機能）

ストリーク計算
- 連続記録日数を計算
- 違反日は除外して計算
- 最長連続日数記録

フェーズ（称号）システム
- Rookie (0日)
- Carnivore (7日)
- Hunter (30日)
- Predator (90日)
- Apex (180日)
- Legend (365日)
- 次のフェーズまでの日数表示でモチベーション向上

カレンダー表示
- 記録をカレンダー形式で可視化
- 違反日の表示

7. 履歴管理

HistoryScreen
- 過去の日次ログを表示（新しい順）
- アコーディオン形式の詳細表示
- 削除機能（ゴミ箱アイコン、確認ダイアログ）
- Argument Card統合（栄養素メトリクスタップ）
- Recovery Protocol表示
- 空状態表示

統計・分析
- 栄養素トレンドグラフ
- 体重推移グラフ
- 週間・月間サマリー

8. その他の機能

日記機能
- メンタル・体調・身体能力など総合的に記録
- Detective AIによる日記と食生活の因果関係分析

多言語対応
- 英語、フランス語、ドイツ語、日本語、中国語
- 言語設定画面から切り替え可能

データ管理
- データエクスポート機能（GDPR対応）
- データ削除機能
- プライバシーポリシー・利用規約

認証機能（オプション）
- Supabase連携によるクラウドバックアップ
- 複数デバイス間での同期
- 認証なしでもlocalStorageで動作可能

ドット絵UI
- アプリ全体をドット絵スタイルに変更可能
- 設定画面から切り替え可能

技術的な特徴

カーニボア基準に基づく栄養計算

タンパク質
- 体重1kgあたり1.2-2.2g（活動レベルに応じて調整）
- 動物性タンパク質と植物性タンパク質を区別
- バイオアベイラビリティを考慮した有効タンパク質計算

脂質
- タンパク質の1.5-2倍（高脂質推奨）
- オメガ3/6比率を追跡
- 炎症スコアの計算

炭水化物
- 実質ゼロ（植物性食品は検知対象）
- 正味炭水化物の計算
- 食物繊維の追跡

微量栄養素
- カーニボア医者の推奨値に基づく
- 動的必要量の計算（性別、年齢、活動レベル等を考慮）
- ビタミンC: 目標値0mg（糖質ゼロ環境下では不要）
- ナトリウム: 目標値5000mg以上（低インスリン状態では高用量が必要）
- マグネシウム: 目標値600mg（土壌枯渇により肉のみでは不足しがち）
- カリウム: 目標値4500mg（ナトリウム摂取増に対応）

科学的根拠の重視
- カーニボア専門家の主張に基づく栄養目標値
- エビデンスベースの機能設計
- 推測ではなく、確立された知見を優先
