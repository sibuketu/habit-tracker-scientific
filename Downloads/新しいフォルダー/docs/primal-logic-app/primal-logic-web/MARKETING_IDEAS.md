# Primal Logic - マーケティング案（メモ）

## ターゲット層

### 1. カーニボアダイエットに懐疑的な人
**キャッチコピー案:**
- 「肉しか食べないCarnivoreが怪しいと思う方ほど、使ってほしいアプリ」
- 「カーニボアが本当に栄養的に問題ないか、科学的に検証できる」

**訴求ポイント:**
- バイオアベイラビリティ（吸収率）を考慮した正確な栄養計算
- 植物性栄養素の「見かけ上の数値」ではなく、「実際に使える栄養素」を表示
- エビデンスベースのArgument Cardsで「なぜカーニボアが機能するか」を説明

### 2. 既にカーニボアを実践しているが、微調整が面倒な人
**キャッチコピー案:**
- 「肉食してる人で、微調整がめんどくさい人向け」
- 「毎日の栄養バランス、自動計算で最適化」

**訴求ポイント:**
- 巨大アイコン5つで「反射的に」食品を登録
- カーニボアサイズ（300g）をデフォルトに設定
- 警告ロジックの反転（脂質・塩分が多い→緑）で「肉を食べるたびに怒られない」

### 3. もっと肉食について知りたい人
**キャッチコピー案:**
- 「カーニボアの科学的根拠を知りたい人へ」
- 「進化生物学×生化学で解き明かす、肉食の真実」

**訴求ポイント:**
- Argument Cards（論破カード）で「なぜカーニボアが正しいか」を3段階で説明
- AIアシスタントがカーニボア専用のアドバイスを提供
- Deep Research機能（将来実装）で最新の研究を参照

## 広告コピー案


### ロングバージョン（ブログ・記事用）
```
「カーニボアダイエット」を実践している、または興味がある方へ。

既存の栄養アプリ（MyFitnessPalなど）は、「カロリー至上主義」や「脂質＝悪」というロジックが前提になっています。

しかし、カーニボアダイエットでは：
- 脂質は必須のエネルギー源（Good）
- 塩分は電解質維持に必須（Good）
- カロリーではなく、P:Fバランス（タンパク質と脂質の比率）を重視

Primal Logicは、カーニボアの生理学（Primal Logic）に基づき、ユーザーのメンタルモデルに合致したUXを提供します。

【主な機能】
1. Anxiety-Free Gauges: 肉を食べるたびに「警告」が出ない
2. Argument Cards: 「なぜカーニボアが正しいか」を科学的に説明
3. バイオアベイラビリティ計算: 植物性栄養素の「見かけ上の数値」を排除
4. AIアシスタント: カーニボア専用のアドバイスを提供

「肉しか食べない」って本当に大丈夫？と疑問に思う方ほど、使ってみてください。
```
