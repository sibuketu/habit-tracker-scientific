## アプリ名について

**確認が必要**: 
- コード内では「Primal Logic」が使われている
- 画像生成のプロンプトでは「CarnivoreGuid」が使われている
- どちらが正しいか確認が必要

**推奨**: 実際のアプリ名を確認してから送信

## SNSとアプリの情報源について

**同じ情報源を使用**:
- アプリのAIチャット: カーニボア専門家の情報を統合
- 動画生成: 同じカーニボア専門家の情報をベースにスクリプト生成
- アプリのAIが嘘情報に反論する機能も実装済み

**つまり**: アプリと動画コンテンツは同じ情報源（カーニボア専門家の情報）を使用している

## コンテキスト禁止について

**提案**: コンテキスト禁止はやめるべき

**理由**:
- コンテキスト（Rules、仕様書、既存コード）は重要な情報源
- ただし、コンテキストに依存しすぎず、実際のコードやファイルを確認する
- コンテキストと実際のコードが矛盾する場合は、実際のコードを優先

**推奨**: 
- コンテキストを参考にするが、実際のファイルを必ず確認する
- コンテキストと実際のコードが矛盾する場合は、実際のコードを優先
- ユーザーが「コンテキスト禁止」と言った場合は、その指示に従う
