# ビタミンCを増やす方法（カーニボア版）

## カーニボアでのビタミンCの考え方

カーニボアダイエットでは、**炭水化物を摂取しないため、ビタミンCの必要量が大幅に減ります**。

### なぜ必要量が減るのか？

**Glucose-Ascorbate Antagonism（グルコース-アスコルビン酸拮抗）理論:**
- 炭水化物（グルコース）が多いと、ビタミンCの必要量が増える
- 炭水化物が少ない（ケトーシス）状態では、ビタミンCの必要量は**約10mg/日**で十分
- 通常の推奨量（100mg/日）は、高炭水化物食を前提にしている

### カーニボアでのビタミンC必要量

- **炭水化物 < 20g**: 10mg/日
- **炭水化物 20-100g**: 10mg + (炭水化物g × 0.4)mg
- **炭水化物 > 100g**: 100mg/日

## ビタミンCを増やす方法（動物性食品）

### 1. 内臓肉（特にレバー）

- **牛レバー**: 1.3mg/100g
- **鶏レバー**: 13mg/100g（高め）
- **豚レバー**: 23mg/100g（非常に高い）

**推奨**: 週に2-3回、50-100gのレバーを食べる

### 2. 生肉（加熱しない）

- 生の肉にはビタミンCが含まれているが、加熱で失われる
- **注意**: 生肉は食中毒のリスクがあるため、信頼できるソースから購入すること

### 3. 生卵

- 卵黄には少量のビタミンCが含まれる
- 加熱すると失われる

### 4. 生魚（刺身）

- サーモン、マグロなどに少量含まれる

## 実践的なアドバイス

### 通常のカーニボア食（炭水化物 < 20g）

**必要量: 10mg/日**

- 通常の肉食だけでも十分な場合が多い
- レバーを週に1-2回食べれば確実

### ケトボア（少量の炭水化物を摂取）

**必要量: 10-50mg/日（炭水化物量による）**

- レバーを週に2-3回食べる
- または、生の肉・魚を少量食べる

### ビタミンCが不足している場合

1. **レバーを増やす**: 週に3-4回、50-100g
2. **生肉を試す**: 信頼できるソースから（リスクあり）
3. **サプリメント**: 動物性のビタミンCサプリ（通常は不要）

## 注意事項

⚠️ **植物性食品は推奨しません**
- カーニボアの原則に反する
- 植物性のビタミンCはバイオアベイラビリティが低い

⚠️ **過剰摂取の心配は不要**
- ビタミンCは水溶性で、余分は排出される
- ただし、通常のカーニボア食では過剰になることはほぼない

## まとめ

カーニボアでは**ビタミンCの必要量が非常に少ない**ため、通常の肉食だけでも十分です。不足が心配な場合は、**レバーを週に2-3回食べる**だけで解決します。
