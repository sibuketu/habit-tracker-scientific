# 価格戦略の詳細解説

## 「実質的な価格」とは

### 定義

**実質的な価格** = ユーザーが実際に支払う金額

**計算式**:
```
実質的な価格 = 参照点（$29.99） - Gift機能による割引 - クーポンによる割引
```

**例**:
- 参照点: $29.99/月
- Gift機能による割引: $10（既存ユーザーが新規ユーザーにGiftを送った場合）
- クーポンによる割引: $5（50%オフクーポンを使用した場合）
- **実質的な価格**: $29.99 - $10 - $5 = $14.99/月
